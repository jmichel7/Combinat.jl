# Combinat
Combinatorics and number theory.

This  module  started  as  a  Julia  port of combinatorics and basic number
theory functions from GAP. The only dependency is the `Primes` package. See
the  docs for  the list  of functions  and comments  on how  it compares to
`Combinatorics.jl`.
